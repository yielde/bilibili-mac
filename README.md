# bilibili-mac

一款基于C++ QT的跨平台软件（UI毛坯风格），PS：主要是mac没找到类似win直播姬的客户端（windows直接下载直播姬就好啦）
实现功能：

- [x] 扫码登录
- [x] 选取直播区，开播获取推流码
- [x] 关闭直播
- [ ] 弹幕窗口
  - [ ] 接收弹幕
  - [ ] 发送弹幕





# 框架

1. QT6.6.2

2. C++版本>=11
3. QR-Code-generator：https://github.com/nayuki/QR-Code-generator
